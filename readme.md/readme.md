## Стек технологий
* **Backend:** PHP 8+
* **База данных:** MySQL
* **Frontend:** HTML5, CSS3, JavaScript (Fetch API)
* **Среда разработки:** MAMP / XAMPP / OpenServer

## Инструкция по запуску проекта

Для корректной работы веб-приложения на локальном компьютере необходимо выполнить следующие шаги:

1. **Подготовка сервера:**
   * Установите и запустите локальный веб-сервер.
   * Поместите всю папку с проектом `flowstate` в корневую директорию сервера.

2. **Настройка базы данных:**
   * Откройте интерфейс `phpMyAdmin`.
   * Создайте новую пустую базу данных с именем `flowstate` и кодировкой `utf8mb4_0900_ai_ci`.
   * Импортируйте в эту базу дамп таблиц или выполните SQL-запросы для создания таблиц `users`, `teachers`, `workouts`, `reviews`, `tariffs`.

3. **Подключение к БД:**
   * Откройте файл конфигурации `db.php` в корне проекта.
   * Установите актуальные настройки подключения к вашей локальной базе данных:
     ```php
     $host = 'localhost';
     $db   = 'flowstate';
     $user = 'root'; 
     $pass = 'root';
     ```

4. **Запуск:**
   * Откройте браузер и перейдите по адресу: `http://localhost:8888/flowstate/index.php`.
   * Для входа в панель администратора перейдите по адресу `http://localhost:8888/flowstate/Admin.php`.
